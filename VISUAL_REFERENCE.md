## 🌈 COLOR USAGE

**Brass Theme (Default):**
- Background: Deep warm charcoal
- Cards: Slightly lighter charcoal
- Text: Warm off-white
- Accents: Brass/gold (#c9a961)

**Ivory Theme (Alternate):**
- Background: Cool charcoal
- Cards: Slightly lighter cool gray
- Text: Cool off-white
- Accents: Ivory/cream (#e8dfc8)

---

## ✨ INTERACTIVE STATES

### Links
- Default: Brass color
- Hover: White color
- Focus: Brass outline

### Cards
- Default: Static
- Hover: Lift up 2px, brass border glow

### Buttons
- Default: Brass background
- Hover: Transparent with brass border
- Focus: Brass outline

### Mobile Menu
- Closed: ☰ icon
- Open: ✕ icon
- Animation: Smooth slide-down
